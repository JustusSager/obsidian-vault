# Modes
|           |       |
| --------- | ----- |
| normal    | `ESC` |
| insertion | `i`   |
| visual    | `v`   |

# Im normal mode
|                |                    |
| -------------- | ------------------ |
| `dd`           | eine Zeile löschen |
| `:q`           | vim verlassen      |
| `:w`           | Datei speichern    |
| `/SUCHBEGRIFF` | suche              |

# Im visual mode
|     |              |
| --- | ------------ |
| `d` | Ausschneiden |
| `y` | Kopieren     |
| `p` | Einfügen     |
